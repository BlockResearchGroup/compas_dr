********************************************************************************
Installation
********************************************************************************

Stable
======

Stable releases can be installed from PyPI.

.. code-block:: bash

    pip install compas_dr


Development
===========

The latest development version can be installed from GitHub.

.. code-block:: bash

    git clone https://github.com/bleockresearchgroup/compas_dr.git
    cd compas_dr
    pip install -e ".[dev]"
